# Usage
<!-- usage -->
```sh-session
$ npm install -g orchestrator
$ orchestrator COMMAND
running command...
$ orchestrator (-v|--version|version)
orchestrator/0.0.0 darwin-x64 node-v10.12.0
$ orchestrator --help [COMMAND]
USAGE
  $ orchestrator COMMAND
...
```
